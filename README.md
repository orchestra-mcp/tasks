# @orchestra-mcp/tasks

Task management UI components for Orchestra MCP

Part of the [Orchestra MCP](https://orchestra-mcp.dev) ecosystem.

## Install

```bash
npm install @orchestra-mcp/tasks
# or
pnpm add @orchestra-mcp/tasks
```

## Usage

```tsx
import { } from '@orchestra-mcp/tasks'
```

## Requirements

- React >= 18.0.0
